Data Input

Model yang diterapkan pada aplikasi menggunakan dua jenis data input, yaitu Data APBD dan Nomenklatur akun.

Standardisasi

Standarisasi Nomenklatur akun dengan mengkombinasikan pre-trained model BERT dan metode pencarian text semantic search berdasarkan ukuran cosine similarity.

Analisis

Analisis terbagi menjadi dua tahap, yaitu tahap Clustering berdasarkan kemiripan alokasi dana dan Prediksi dengan target variabel IPM untuk melihat dampak dari setiap komponen alokasi terhadap kemajuan daerah.
